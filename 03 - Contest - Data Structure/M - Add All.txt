M - Adicionar todos

Sim!! O nome do problema reflete sua tarefa; basta adicionar um conjunto de números. Mas você pode se sentir condescendente ao escrever um programa C / C ++ apenas para adicionar um conjunto de números. Esse problema simplesmente questiona sua erudição. Então, vamos adicionar um pouco de criatividade.

A operação de adição exige custo agora, e o custo é a soma dos dois a serem adicionados. Portanto, para adicionar 1 e 10, você precisa de um custo de 11. Se você deseja adicionar 1, 2 e 3. Existem várias maneiras

1 + 2 = 3, custo = 3
3 + 3 = 6, custo = 6
Total = 9

1 + 3 = 4, custo = 4
2 + 4 = 6, custo = 6
Total = 10

2 + 3 = 5, custo = 5
1 + 5 = 6, custo = 6
Total = 11

Espero que você já tenha entendido sua missão, adicionar um conjunto de números inteiros para que o custo seja mínimo.

ENTRADA

Cada caso de teste começará com um número positivo, N (2 <= N <= 5000), seguido por números inteiros positivos (todos são menores que 100000). A entrada é finalizada por um caso em que o valor de N é zero. Este caso não deve ser processado.

SAÍDA

Para cada caso, imprima o custo total mínimo da adição em uma única linha.

EXEMPLO DE ENTRADA
3
1 2 3
4
1 2 3 4
0

EXEMPLO DE SAÍDA
9
19
